TranslatorBot
Telegram-бот переводчик
Данный проект представляет собой Telegram-бота, предназначенного для перевода текста на разные языки прямо в мессенджере Telegram.

Актуальность
Боту не нужно устанавливать отдельные приложения для перевода. Пользователь получает перевод прямо в чате, что экономит время.

Функции бота
Перевод текста на разные языки
Простое взаимодействие с пользователем
Работа через команды Telegram
Используемые технологии (зависимости)
Python
Telegram Bot API
Библиотека telebot, requests
Установка и запуск
Установить зависимости: `bash pip install -r requirements.txt
Инструкция
Для начала работы бота нужно ввести команду для запуска (/start)
Затем нужно выбрать с какого языка будет рализовываться перевод (их 5) Русский, English, Français, Deutsch, Español.
После этого нужно выбрать на какой язык будет осуществляться перевод.
Нужно ввести текст который хотите перевести.
Бот возвращает переведённый текст пользователю.
